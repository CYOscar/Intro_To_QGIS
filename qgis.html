QGIS Tutorial

Download QGIS and Set Up Project

Follow these steps:

  1. Download the latest version of QGIS: QGIS Download
  2. Open QGIS and create a new project:

    3. Once you create a project, you will see this page.

Add Data:

If data format is CSV, convert CSV to points on QGIS.

Go to the Browser tab on the left to look for the folder where you store your data.

PLEASE NOTE: You will want to make sure your data layers are saved with the same CRS as your project in order for them to show up properly in QField. If you want to use a basemap, it is best to set your project CRS to EPSG: 3857 (WGS 84/Pseudo-Mercator) since popular basemaps you can access through QuickMapServices in QGIS use this CRS.


To edit symbology of the layer >> double-click the layer on the left >> go to Symbology tab where you can make changes to the appearance of the layer

Add basemaps:

Go to plugins on top menu >> Manage and install plugins

  1. In the Plugins Window, search for QuickMapServices then click Install button. Note if you have this plugin installed from a previous version of QGIS then it may have been updated as well.
  2. After it installs, you should be able to find the QuickMapServices button in the Web Toolbar.
  3. Click on the button to get the drop down menu >> Settings. In the QuickMapServices Settings window, go to the More Services tab >> Click Get Contributed Pack to get more basemaps. Then go to the Visibility tab and turn on/off the basemaps you want to display in the menu
  4. You will find basemaps on a globe image of QuickMapServices. Add a basemap of your choice to the map.

Install QFieldSync plugin:

  1. Go to plugins on top menu >> Manage and install plugins
  2. Search ‘QFieldSync’
  3. Click ‘install’
  4. After installation, you should see ‘QFieldSync’ on the plugins menu.
